# Void Clicker - Complete Setup Guide

## Folder Structure
```
void-clicker/
├── package.json
├── vite.config.js
├── tailwind.config.js
├── postcss.config.js
├── index.html
└── src/
    ├── main.jsx
    ├── App.jsx
    └── index.css
```

## Setup Instructions

1. Create the folder:
```bash
mkdir void-clicker
cd void-clicker
```

2. Create all the files below (copy each file's content)

3. Install dependencies:
```bash
npm install
```

4. Run the app:
```bash
npm run dev
```

5. Open browser to http://localhost:5173
